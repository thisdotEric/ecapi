# ecapi

## Steps to run

1. Install dependencies.

```
  yarn install
```

2. Create .env file on root directory and fill up the following:

```
  NODE_ENV=development
  PORT=3000
  MONGO_DB=
  ACCESS_TOKEN_SECRET=accesstoken
  REFRESH_TOKEN_SECRET=refreshtoken
```

3. Run the unit test

```
  yarn test
```

4. Run the application.

```
  yarn dev
```
